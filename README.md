# Exercícios de Estruturas em C++

## Descrição
Este repositório contém exercícios básicos em C++ destinados ao aprendizado e prática de estruturas de controle fundamentais, especialmente **condicionais** e **laços `for`**. Cada arquivo demonstra um conceito por meio de exemplos simples e diretos.

## Exercícios incluídos

### 1. Estruturas Condicionais (`if`, `else if`, `else`)
Exercícios voltados para a tomada de decisão no código.  
Entre os exemplos presentes:
- Identificação de números positivos, negativos ou iguais a zero.  
- Comparação entre variáveis para determinar maior, menor ou igualdade.  
- Seleção de ações com base em faixas de valores.

### 2. Laço `for`
Atividades que utilizam o laço `for` para repetir instruções por um número definido de vezes.  
Incluem:
- Impressão de sequências numéricas.  
- Contagens crescentes e decrescentes.  
- Execução de cálculos simples dentro do loop, como somatórios ou multiplicações.

### 3. Exercícios combinados
Alguns exercícios utilizam condicionais dentro de laços `for`, reforçando lógica estruturada e organização do código.

## Objetivo pedagógico
- Praticar lógica condicional e controle de repetição.  
- Servir como base para iniciantes em C++.  
- Demonstrar na prática como estruturas fundamentais funcionam em programas reais.
